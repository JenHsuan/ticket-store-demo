# Ticket store demo

## Technical stacks
* React
* Redux
* React hooks
* Redux hooks
* Redux thunk
* Styled component
* Styled system
* json-server

## Demo

## Usages
```
git clone https://github.com/JenHsuan/ticket-store-demo.git
cd ticket-store
npm install
npm start
```
